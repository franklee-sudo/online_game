# Online Game (Web Survival)

这是一个基于浏览器的生存类网页游戏项目，支持中文与英文入口。

## 在线地址
- 中文游戏：`https://game.yoyolab.top/`
- 英文游戏：`https://game.yoyolab.top/en/play.html`

## 主要玩法
- WASD/方向键移动
- 自动攻击与武器进化
- 掉落拾取与升级三选一
- Boss 预警与多阶段弹幕战
- 宝箱事件（法球风暴 / 清屏）
