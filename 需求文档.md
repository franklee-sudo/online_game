# 网页生存游戏需求文档（公开版 v1.0）

## 1. 项目定位
- 类型：2D 网页生存游戏（浏览器即开即玩）。
- 目标用户：喜欢 Vampire Survivors/雷电式“高压弹幕 + 成长爽感”的玩家。
- 核心卖点：自动攻击、武器进化、宝箱随机事件、阶段 Boss 战。

## 2. 核心玩法（公开）
1. 玩家移动走位（WASD/方向键）。
2. 自动攻击最近敌人。
3. 击杀掉落，靠近自动拾取。
4. 升级三选一强化，构建流派。
5. 固定节点触发 Boss 预警并进入 Boss 战。

## 3. 关键功能
- 职业选择（至少 3 职业）。
- 武器系统（魔法弹、回旋镖、闪电等）。
- 技能进化（单发 -> 多发 -> 溅射/连锁）。
- 宝箱事件：
  - 法球风暴（限时多方向发射）
  - 清屏效果（保留战斗节奏）
- 暂停机制：`ESC` 打开/恢复暂停。

## 4. 内容与版本方向
- 中英文双语入口：
  - 中文：`/index.html`
  - 英文：`/en/play.html`
- 运营与推广基础页面：
  - 联系方式、隐私政策、服务条款（中英双语）

## 5. 公开验收标准（DoD）
- 游戏流程完整：菜单 -> 职业 -> 战斗 -> 升级 -> 结算。
- 自动攻击、掉落拾取、升级生效稳定。
- Boss 具备预警与多类型弹幕机制。
- 中英文页面均可访问，核心信息一致。
